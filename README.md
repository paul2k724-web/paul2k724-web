<details>
<summary><sub><b>Enable the snake</b> — add this as <code>.github/workflows/snake.yml</code>, then run it once</sub></summary>

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:

permissions:
  contents: write

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Important: the snake will show only after this workflow creates the `output` branch. If the Action fails with permission denied, set **Settings → Actions → General → Workflow permissions → Read and write permissions**.

</details>
